1. Словарь (Dictionary)

Хранение слов / выражений (DictionaryEntry).
Поддержку:
нескольких значений (senses),
переводов,
примеров,
изображений,
произношений.

Явное происхождение данных (sourceSlug).
Редактирование контента
Cоздание и удаление слов.

2. Изучение слов (Study / SRS)

Поддерживается:

Интервальное повторение (SRS).
Четкое состояние карточки:
NEW → LEARNING → REVIEW → MASTERED.
История повторений.
Очередь на повтор (studyQueue).

Осознанное ограничение:
Карточка привязана к слову целиком, не к sense.
 
3. Inbox / Capture Flow

Поддерживается:
Быстрое сохранение слов / контекста без принятия решений.
Явная конвертация inbox → word / example.

4. Suggestions (внешние источники)
Поддерживается:
Stateless получение данных.
Несколько источников.
Частичный успех.
Ручной контроль пользователем.
