# my-english

В этом проекте я хочу создать свой личный сервис для отслеживания своего прогресса изучения английского языка.

Для начала я хочу отслеживать то, какие слова я ууже знаю а какие нет. MVP должно выглядеть следующим образом:

1. Собираем датасет слов английского языка, которые разбиты по уровням и по категориям. Для каждого слова хочется отслеживать:
    1. Само слово
    2. Как часто оно используется
    3. В каком контексте оно использууется
    4. Определение этого слова на английском.
    5. Определение на русском
    6. [Какие еще поля стоит добавить?]

2. Делаем две страницы: 
    1. Отслеживание слов: просто два столбца: заученные и незаученные. Для каждого столбца можно приделать фильтры и сортировки
    2. Заучивание новых слов. Пока что просто добавляем слова в ручном формате. Поле для ввода с подсказками для ввода.
